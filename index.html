My Projects

Review App

project repo

This project is a review web-app that allows an authenticated user via google to leave a review. The user review field is then translated to a specified language using google translate API. When visiting the site it allows anyone to see all the reviews meaning that you don’t need to be authenticated. The app uses dockerfile to create a container image which is then stored on google container registry and run using cloud run. The app is written in python, uses flask as the web framework, and cloud datastore as the database. This project was built for my final project for Internet, Web and Cloud systems course, so it’s a simple web-app design which focused more on the cloud features and the back-end implementations. The project was constructed using the model-view-presenter architectural pattern.

SRE Project

project repo

This project comes from the SRE (site reliability engineering) course that I took at PSU. It was a team project, so I worked on it with a teammate. This project focused more on making a mock site reliable. The professor monitored the site 24/7 and did stress tests to see if our site withstands. In order to improve our reliability, we used Docker and Docker-compose since we split up our components into different images. Then we used Kubernetes to create bootable VMs within gcloud infrastructure to be able to scale when demand is high and scale down when demand is low; thus, reducing the project cost and improving peak time performance. We also used Cloudflare to create a tunnel for each ephemeral virtual machine to be able to load balance efficiently and prevent denial of service attacks.
